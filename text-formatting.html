Text formatting di HTML

Text Tebal di HTML

Teks formatting itu penting! Karena dapat membuat tulisan terlihat lebih menarik sehingga akan membuat pengunjung senang membacanya. Gunakan teks miring untuk memberikan penekanan pada text, sehingga akan menarik perhatian pembaca. Biasanya digunakan pada istilah asing atau kata serapan dari bahasa daerah. Text formatting itu penting, karena dapat membuat text terlihat lebih menarik dibandingkan text biasa plain text. Rumus luas persegi adalah S2, di mana S adalah sisi dari persegi. Lalu O2 adalah rumus kimia dari oksigen. Marker biasanya digunakan untuk menandai bagian text yang penting. Kalau di dunia nyata, kita menggunakan stabilo untuk membuat marker. Perintah javasript untuk menampilkan text console adalah console.log(). Kita juga bisa menampilkan isi variabel dengan fungsi ini. Misalkan kita punya variabel name, maka kode programnya bisa ditulis seperti ini: 

        var name = "Petnasi Kode";
        console.log(name);
Maka hasil outputnya: Petani Kode Untuk melakukan ulang program, lakukan refresh dengan menekan tombol F5 Penggabungan format text bisa dilakukan dengan menuliskan tag-tag yang akan dipakai. Misalkan tebal dan garis lurus, maka kita tinggal pakai tag b dan tag u.
